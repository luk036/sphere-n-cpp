auto lds2::CylindN::pop ( ) -> vector< double >

Generate the next point using cylindrical coordinate method.

Generates a uniformly distributed point using cylindrical coordinates. This method uses the Van der Corput sequence to generate the z-coordinate and recursively generates the base dimensions, then combines them using cylindrical coordinate transformation.
